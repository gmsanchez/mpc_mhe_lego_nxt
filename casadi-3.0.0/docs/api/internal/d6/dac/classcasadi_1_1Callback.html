casadi::Callback Class Reference

Callback function functionality This class provides a public API to the FunctionInternal class that can be subclassed by the user, who is then able to implement the different virtual method. Note that the Function class also provides a public API to FunctionInternal, but only allows calling, not being called. More...

#include <callback.hpp>

Detailed Description

The user is responsible for not deleting this class for the lifetime of the internal function object.

KernelSum computes

F(V,X) = sum_i sum_j f ( [i;j], V(i,j), X)

with X: [x;y]
