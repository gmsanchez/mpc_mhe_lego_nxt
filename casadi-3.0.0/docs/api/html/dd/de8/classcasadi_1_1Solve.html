casadi::Solve< Tr > Class Template Reference

An MX atomic for linear solver solution: x = r * A^-1 or x = r * A^-T. More...

#include <solve.hpp>

Detailed Description

template<bool Tr>
class casadi::Solve< Tr >

Forward derivatives: x_dot = (r_dot - x * A_dot) * A^-1

Adjoint derivatives: r_bar = x_bar * A^-T A_bar = -x^T * r_bar
