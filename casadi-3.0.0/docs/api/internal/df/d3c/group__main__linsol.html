Main_linsol

Create a solver for linear systems of equations Solves the linear system A*X = B or A^T*X = B for X with A square and non-singular

If A is structurally singular, an error will be thrown during init. If A is numerically singular, the prepare step will fail.

The usual procedure to use Linsol is:

  1. init()
  2. set the first input (A)
  3. prepare()
  4. set the second input (b)
  5. solve()
  6. Repeat steps 4 and 5 to work with other b vectors.

The standard evaluation combines the prepare() and solve() step and may therefore more expensive if A is invariant.
